Documentacion del Servicio

Referencia tecnica y estructurada del servicio de generacion de cartas de invitacion para inmigracion mexicana. Esta pagina esta pensada tanto para desarrolladores como para agentes de inteligencia artificial.

Descripcion General

Servicio
Generacion automatizada de cartas de invitacion en PDF
Formato de salida
PDF formal para inmigracion mexicana (INM)
Idioma
Espanol (es-MX)
Tiempo de entrega
Instantaneo (generacion automatizada)
Entrega
Descarga PDF inmediata + envio por email
Procesador de pago
Stripe

Tipos de Documento

carta_invitacion_turismo

Carta de invitacion para turismo. Se utiliza para visitas personales: reuniones familiares, turismo, visitar a la pareja, eventos culturales, viajes por tratamientos medicos.

carta_invitacion_negocios

Carta de invitacion para negocios. Se utiliza para visitas profesionales: conferencias, ferias comerciales, reuniones de negocios, actividades profesionales no remuneradas.

Motivos de viaje soportados: turismo, negocios, estudios, actividades no remuneradas, transito, tratamientos medicos.

Precios

Plan Esencial

$5 USD

  • Un solo viajero
  • Itinerario basico
  • Datos del anfitrion y viajero
  • Relacion, motivo y alojamiento
  • Fechas de viaje
Crear carta

Plan Completo

$9 USD

  • Multiples viajeros (acompanantes)
  • Itinerario multi-destino
  • Transporte entrada/salida detallado
  • Desglose de responsabilidad financiera
  • Medios de transporte en Mexico
Crear carta
